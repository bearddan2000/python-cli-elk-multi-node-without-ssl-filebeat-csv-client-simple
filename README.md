# python-cli-elk-multi-node-without-ssl-filebeat-csv-client-simple

## Description
Reads a csv file into fikebeat then logstash which feeds a multi node for data in `dog-demo` document.

## Tech stack
- python
- elasticsearch
- kibana
- logstash
- filebeat

## Docker stack
- python
- elasticsearch
- kibana
- logstash
- filebeat

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
